    public class SumaDosNumeros {
        public static void main(String[] args) {
            Scanner input = new Scanner(System.in);
            int num1, num2, resultado;

            System.out.print("Ingrese el primer número: ");
            num1 = input.nextInt();

            System.out.print("Ingrese el segundo número: ");
            num2 = input.nextInt();

            resultado = num1 + num2;

            System.out.println("El resultado de la ses: " + resultado);
        }
    }

glosario

  • Algoritmo: Conjunto ordenado de operaciones sistemáticas que permite hacer un cálculo y hallar la solución de un tipo de problemas.
  • Variable: Una variable es un objeto del lenguaje cuyo valor se puede cambiar. Antes de utilizar una variable ésta debe de ser declarada. Al declarar una variable, se le asocia un identificador, es decir, un nombre, con un tipo de almacenamiento cuya forma determina la visibilidad y existencia de la variable.
  • Tipo de dato: El tipo de la variable nos indica el conjunto de valores que puede tomar y las operaciones que pueden realizarse con ella. Unas de las mas utilizadas son

    • int

      Int indica un tipo de dato que almacena numeros enteros, pueden ser 0, -2, 3...

    • float

      Un dato Float indica un tipo de dato que almacena numeros decimales (tambien se le dice "doubles" con siertas diferencias), pueden ser 0.2, 0.5, -0.1...

      Dato: si a un float se le pasa un numero entero (9) lo convierte implicitamente en (9.0), por lo cual se pueden usar enteros en datos de tipos float pero no datos decimales en tipos Int

    • boolean

      Un dato boolean indica que un dato sera verdadero o falso, como ejemplificación en la vida real podemos hacer un boolean cuando se enciende un foco, solo hay 2 opciones, encendido=verdadero o encendido=falso

    • String

      El string es una cadena de texto, la forma en la que son representados son con los "" indican en si un texto, puede ser "Hola" "abc123" "Nombre" "país"

      Dato: El string tambien peude usar simbolos como () & / % $ ^^, etc.

  • Palabra reservada: En los lenguajes informáticos, una palabra reservada es una palabra que tiene un significado gramatical especial para ese lenguaje y no puede ser utilizada como un identificador de objetos en códigos del mismo, como pueden ser las variables. Por ejemplo las palabra reservada en Pseint pueden ser Escribir Leer Algoritmo, y no se pueden usar para crear variables.
